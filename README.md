# E-Learning Application

A full-stack E-learning platform with role-based authentication, CRUD operations, and responsive UI.

## Features

- Role-based authentication (Student, Instructor, Admin)
- CRUD operations for users and courses
- RESTful API
- Responsive web UI
- Mobile app (Flutter)
- Basic security practices

## Tech Stack

- Backend: Node.js, Express, MongoDB, JWT
- Frontend: React, Vite
- Mobile: Flutter
- Database: MongoDB

## Setup

### Backend
1. cd backend
2. npm install
3. Set up MongoDB and update .env
4. npm run dev

### Frontend
1. cd frontend
2. npm install
3. npm run dev

### Mobile
1. cd mobile
2. flutter pub get
3. flutter run
